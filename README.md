# Киберсигурност - Лабораторно упражнение 2

## Описание
Проектът съдържа практически задачи по киберсигурност, реализирани на езика C.
Демонстрирани са различни атаки и слабости в криптографски системи.

## Задачи

### Задача 1: Seed Crack
Реализирана е brute-force атака срещу генератор на случайни числа,
инициализиран със srand(time(NULL)).
Атакуващият проверява възможните стойности за последните 600 секунди.

### Задача 2: RC4
Имплементация на RC4 поточен шифър с:
- KSA (Key Scheduling Algorithm)
- PRGA (Pseudo-Random Generation Algorithm)

Криптирането се извършва чрез XOR операция.

### Задача 3: Many-Time Pad Attack
Демонстрира се уязвимост при повторно използване на ключ.
Чрез XOR се възстановява част от съобщение.

## Домашна работа

- RC4 криптиране на файл
- Brute-force възстановяване при ransomware сценарий
